# DunzoMarket-Analysis
Overview:

This repository contains the 1DunzoMarket_Report.pbix, a fully interactive Power BI report created to analyze key performance indicators such as:

1.Sales and revenue trends

2.Order volume analysis

3.Delivery performance

4.Inventory levels

5.Customer behavior & segmentation

6.Category-wise and zone-wise performance

The report is divided into multiple pages focusing on granular insights, making it suitable for decision-makers, data analysts, and BI enthusiasts.
